# learning-management-system
This project is developed as part of a pupil's Web Development 201 course. It serves as a practical application of the concepts and skills learned throughout the course.

Overview
Learning Management System (LMS) is a web-based platform designed to facilitate online learning and training. It provides educators, learners with tools to manage courses, deliver content, assess progress.

Features
-> Educators Feature
  1. Create a course
  2. Add Chapters
  3. Build Pages
  4. View Reports
  5. Delete chapter, course or page
-> Students Feature
  1. Enroll in a course
  2. Mark pages as complete
  3. View Chapter names without enrolling
  4. Show progress status of enrolled courses
-> Common Features
  1. Change Password
